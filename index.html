operateElems

Operations with elements. If 3-d parameter is not set - works like forEach, if set - bing operations to one event or multiple events
WellUI.operateElems(elemsArray, cb, eventsName) @paramelemsArray {Array} Array of elements for handling @paramcb {Function} Callback that will be apply for event when is set or just work when elements will be iterated (when event is not set) !Important: this -> iterated element @parameventsName {String | Array} Event name. If one event, format will be like this: 'click', but when you want use cb for multiple events format will be like this: ['click', 'mouseover']

Do something with each element with 'example' class without event handling

WellUI.operateElems(WellUI.getElemsArr('.example'), function () { this.style.backgroundColor = 'green'; });

You can execute this code into console of this page for a demo.

tip: You can use for create array of elements - getElemsArr.

Do something with each element with 'example' class with one event handling

WellUI.operateElems(WellUI.getElemsArr('.example'), function () { this.style.backgroundColor = 'red'; }, 'mouseover');

You can execute this code into console of this page for a demo.

After execute code in console try hover mouse on method description.

Do something with each element with 'example' class with multiple event handling

WellUI.operateElems(WellUI.getElemsArr('.example'), function () { this.style.backgroundColor = 'red'; }, ['click', 'mouseout']);

You can execute this code into console of this page for a demo.

After execute code in console try click mouse on method description and try out mouse on other non-marked element.

holdEvent

Improved stopPropagation without stopping a bubbling
WellUI.holdEvent = function is return {set, addAndKepEvent, getHoldMarkName} set(e);
Set hold mark to event Object @parame {Event} Event that we want to keep
addAndKepEvent(element, eventName);
set event listener to element and than set hold mark to event Object @paramelement {HTMLElement} Class name that we looking for. Can be with dot like '.className' or clean like 'className' @parameventName {Event} Event that we listen on element
getHoldMarkName()
get hold mark name

Hold event of element with 'ex10' id inside his event handler

var ex10 = document.getElementById('example10'), sub10 = document.getElementById('sub10'), holdEvent = WellUI.holdEvent();

sub10.addEventListener('click', function (e) { if(e[holdEvent.getHoldMarkName()]) { console.log("detect hold click-event") } console.log('normal click-event handle') });

ex10.addEventListener('click', function (e) { holdEvent.set(e); console.log('clicked on the ex10') });

You can execute this code into console of this page for a demo.

When you click on holdEvent title ("ex10"), you will see that we detected bubbled click-event, and we can do something (or abort some operations) and after that event just bubbling to upward
